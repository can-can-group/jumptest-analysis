# Output directory

Generated by scripts (no git tracking for large outputs).

## detection_manifest.json / detection_manifest.csv

Produced by:

```bash
PYTHONPATH=. python script/export_detection_manifest.py [--filter-hz 10] [--limit N]
```

- **detection_manifest.json**  
  Full detail per test: `test_id`, `type`, `status`, `order_ok`, `order_error`, `events` (movement_onset, min_force, take_off, landing with index/time_s/value_N), `metrics` (p1_peak, p2_peak), `points` (key_points list), `key_points_count`.

- **detection_manifest.csv**  
  One row per test. Columns: `test_id`, `type`, `status`, `order_ok`, `order_error`, then for each of onset, min_force, p1, p2, take_off, landing: `*_idx`, `*_time_s`, `*_value_N`, plus `key_points_count`.

Use these to verify that **order** (onset < min_force < P1 < P2 < take_off < landing) and **index/time/value** are correct. Sort by `order_ok` or `key_points_count` to find tests that need algorithm fixes.
